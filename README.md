# CarnivalHub 3D

# Setup Instructions
1. Navigate to the Frontend Directory
``` cd frontend ```
2. Install Dependencies
``` npm install ```
3. Start the Frontend
``` npm run dev ```
4. The frontend will start on http://localhost:5173
